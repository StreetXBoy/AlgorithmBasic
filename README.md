java string 和 int 互相转换

string  转 int :

1、 int i = Integer.parseInt([String]); 或 i = Integer.parseInt([String],[int radix]);

int  转  string :

1、String s = String.valueOf(i);
2、String s = Integer.toString(i);
3、String s = "" + i;
